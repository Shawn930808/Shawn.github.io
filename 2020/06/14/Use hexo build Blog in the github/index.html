1. 安装Homebrew

  • 1.安装
$ /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

  • 2.测试

    $ brew ls   本地软件库
$ brew -v   查看brew版本

    2. 安装Git

  • 1.安装

    $ brew install git

  • 2.测试

    $ git version

    3. 安装 Node.js

  • 1.安装

    $ brew install node

  • 2.测试

    $ node -v
$ npm -v

  • 3.更新

    //单独运行
$ brew update
$ brew upgrade node
$ npm install -g npm
//一条运行
$ brew update && brew upgrade node && npm install -g npm

    4. 安装 hexo

  • 1.安装

    $ npm install -g hexo-cli
    初始化：本地建一个文件夹Hexo，cd命令进入目录，执行以下命令
    $ hexo init
$ npm install

  • 2.本地启动

    $ hexo g
$ hexo s

  • 3.修改主题
    将主题clone到themes文件夹，打开hexo的主目录下的_config.yml文件，找到theme，将值改为主题的名字：

    theme: next

    4. 注册Github，并新建一个仓库（repository）

  • 创建repository：注意名字必须跟前面自己的用户名相同

5. 配置SSH

    1. 生成一个SSH：通过以下命令生成一个ssh_key.pub
      $ ssh-keygen -t rsa -C 你的邮箱

  • 2.点击Github头像，选择settings->SSH and GPG keys
    复制ssh_key.pub里的内容，填进ssh.


6. 将本地内容部署到Github

  • 使用以下命令验证是否与Github链接

    $ ssh -T git@github.com

  • 配置账号

    $ git config --global user.name you username
$ git config --global user.email you email

  • 安装github部署插件

$ npm install hexo-deployer-git --save
  • 打开_config.yml文件，在最底部修改部署配置，注意严格的空格
# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repository: http://github.com/Shawn930808/Shawn930808.github.io.git
  branch: master
  • 保存后在终端执行以下命令
$ hexo clean
$ hexo g
$ hexo d

到此为止，自己的个人网站就成功搭建并部署到Github上了，接下来就可以通过Shawn930808.github.io这个网址访问自己的网站了
